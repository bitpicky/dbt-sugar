# dbt-sugar :candy:

## What is dbt-sugar?

dbt-sugar is a CLI tool that allows users of [dbt](https://www.getdbt.com/) to have fun and ease performing actions around dbt models such as:

- documentation
- test enforcement
  and probably more in a not too distant future.
